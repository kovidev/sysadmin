# 쉽게 서브버전 저장소 만들기 1

- 원격 데스크톱으로 dev.kovi.com에 접속합니다.
- `C:\Users\KOVI\Documents\sysadmin` 으로 이동합니다.
- `createrepo.bat`을 더블클릭합니다.
- 저장소의 이름을 입력합니다.
- svn://dev.kovi.com/{저장소 이름}/trunk 로 접근이 가능한지 확인합니다. 저장소 이름은 이전 단계에서 복사한 복사본의 이름입니다.
